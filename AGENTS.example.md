## A股信号与持仓助手 v6.3

股票池、买入信号、解释原因：

```bash
cd ~/project/a_share_signal_bot && printf '%s\n' '<用户原文>' | ./stockbot_chat.sh
```

触发词：加入、删除、查看股票池、生成买入信号、解释、为什么没买、回测。

盘中持仓交易建议：

```bash
cd ~/project/a_share_signal_bot && ACCOUNT=200000 PORTFOLIO=portfolio.csv ./run_position_monitor.sh
```

触发词：持仓、盘中、交易建议、止损、止盈、减仓、加仓。

规则：

1. 不要自动下单。
2. 不要手工编造股票信号；必须调用脚本。
3. 脚本输出原样回复用户。
4. 如果脚本失败，回复日志最后 120 行。
5. 所有路径使用 `~/project/a_share_signal_bot`。

### 交易计划生命周期新增规则 v6.8

当用户问“买入后怎么办”“后续操作”“止损止盈”“卖出计划”“生成交易计划”“同步交易状态”等，调用：

```bash
cd ~/project/a_share_signal_bot && printf '%s\n' '<用户原文>' | ./stockbot_chat.sh
```

脚本会调用 `trade_manager.py`，用 `trade_state.csv` 记录交易状态，并按主回测一致规则输出：

- T 日信号，T+1 开盘买入计划；
- T+1 买入当天 A 股 T+1 锁定，不提示卖出；
- T+2 起盘中止损、1.5R半仓止盈、3R清仓；
- 收盘跌破 MA20/MA60、大盘弱势，提示下一交易日开盘卖出；
- 当前仍在买入信号中且趋势强时，可给出小步加仓建议。

仍然不能自动下单，必须让用户自行确认执行。
